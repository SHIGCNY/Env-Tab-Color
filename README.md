# 环境标签着色 / Env Tab Color

一个 Chrome / Edge 浏览器扩展（Manifest V3）。按 URL 规则自动识别当前标签页属于 **dev / test / prod** 哪个环境，并用醒目的视觉标记提醒你，避免在生产环境误操作。

## 功能

按 URL 规则把标签页匹配到某个环境后，提供三种可独立开关的标记方式：

| 标记 | 位置 | 说明 |
|------|------|------|
| **页面色条** | 网页内 | 在页面角落显示一个环境色悬浮标签（半透明、点击穿透，不遮挡页面内容）。位置可在四个角中选择。 |
| **标题前缀** | 标签页标题 | 在网页标题前加上 `[ENV]` 前缀，如 `[PROD] 控制台`。 |
| **图标徽章** | 工具栏图标 | 在扩展图标上叠加环境名首字母角标，并用环境色作背景。 |

- **环境**：默认内置 `dev`（绿）/ `test`（橙）/ `prod`（红），可自由增删、改名、改色。
- **规则**：每条规则是一个 URL 模式（支持 `*` 通配符）对应一个环境，从上到下第一条命中生效。
- **配置同步**：所有配置存于 `chrome.storage.sync`，登录同一浏览器账号的设备间自动同步。

## 安装（开发者模式加载）

本扩展未上架商店，通过「加载已解压的扩展程序」方式安装：

1. 下载本仓库到本地（`git clone` 或下载 ZIP 解压）。
2. 打开浏览器扩展管理页：
   - Chrome：地址栏输入 `chrome://extensions`
   - Edge：地址栏输入 `edge://extensions`
3. 打开右上角的 **开发者模式 / Developer mode**。
4. 点击 **加载已解压的扩展程序 / Load unpacked**，选择本仓库**根目录**（包含 `manifest.json` 的那一层）。
5. 安装完成，工具栏会出现扩展图标。

> 修改代码后，回到扩展管理页点该扩展的 **重新加载**，并**刷新**已打开的页面让新逻辑生效。

## 使用

### 快速添加当前页（弹窗）

点击工具栏图标打开弹窗：

- 顶部显示当前页命中的环境（未命中显示「未匹配」）。
- 选择一个环境，点 **添加当前页 →**，会按当前页的 host 自动新增一条规则。
- 勾选 / 取消三种标记方式。
- 点 **打开设置 →** 进入完整配置页。

### 完整配置（设置页）

在设置页可以：

- **标记方式**：开关页面色条 / 标题前缀 / 图标徽章，并选择色条出现的位置（左上 / 右上 / 左下 / 右下）。
- **环境**：增删环境，编辑名称与颜色。
- **规则**：编辑 URL 模式（支持 `*`，如 `*.dev.example.com`）、指定环境、调整优先级（↑/↓，靠上的优先命中）、删除。

## 规则匹配说明

- 模式**不含** `*` 时按**子串包含**匹配（大小写不敏感），如 `test.example.com` 命中 `https://test.example.com/login`。
- 模式**含** `*` 时按通配符匹配，`*` 匹配任意字符，如 `*.dev.*`。
- 多条规则按列表顺序从上到下，**第一条命中**即生效。
- 仅对 `http(s)` 页面生效。

## 项目结构

```
manifest.json        扩展清单（MV3）
src/
  background.js      service worker：维护工具栏图标徽章
  content.js         内容脚本：页面色条 + 标题前缀
  match.js           URL → 环境 的匹配逻辑（纯函数，可单测）
  storage.js         配置读写与默认值
  popup.html/js      工具栏弹窗
  options.html/js    设置页
icons/               扩展图标（16 / 48 / 128）
test/                单元测试
```

## 开发与测试

匹配逻辑有单元测试，用 Node 内置测试运行器（需 Node 18+）：

```bash
node --test test/match.test.js
```

## 权限说明

| 权限 | 用途 |
|------|------|
| `storage` | 保存配置（环境、规则、开关、位置） |
| `tabs` | 读取标签页 URL 以更新图标徽章 |
| `scripting` | 内容脚本注入相关 |
| `<all_urls>` | 需要在任意站点上判断环境并着色 |
